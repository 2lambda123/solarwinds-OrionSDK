Orion.AlertScope

Base type: System.Entity

Summary: This entity allows display list of possible affected objects (scope) for specific alert or all alerts which can affect specific object To be able select some record you have to use WHERE clause with following properties: ResultType - objects/alerts - always required (behaviour depends on it) EntityType - required only for "alerts" mode ObjectId - required only for "alerts" mode AlertId - required only for "objects" mode
